RIGEL(1) Manual de usuario de Rygel RIGEL(1)

NOMBRE

rygel: una colección de servicios AV DLNA/UPnP

SINOPSIS

Rygel [ opciones ]

DESCRIPCIÓN

rygel es una solución multimedia para el hogar que le permite compartir fácilmente audio, video e imágenes, y controlar el reproductor multimedia en su red doméstica. En términos técnicos, es tanto un UPnP AV MediaServer como un MediaRenderer implementado a través de un mecanismo de complemento. La interoperabilidad con otros dispositivos en el mercado se logra mediante el cumplimiento de requisitos muy estrictos de DLNA y la conversión sobre la marcha de los medios al formato que los dispositivos del cliente son capaces de manejar.

OPCIONES

El programa sigue la sintaxis habitual de la línea de comandos de GNU, con opciones largas que comienzan con dos guiones (`-'). Un resumen de las opciones se incluye a continuación.

-h , --ayuda

Mostrar resumen de opciones.

--versión

Muestra la versión del programa.

-n , --network-interface= INTERFAZ

Configure las interfaces de red en las que Rygel escucha. Se puede utilizar varias veces para configurar varias interfaces. INTERFACE puede ser una IP, un nombre de dispositivo o un ESSID (solo Linux).

-p , --port= PUERTO

Configure el puerto de escucha.

-t , --disable-transcodificación

Deshabilite cualquier transcodificación.

-U , --disallow-upload

Deshabilite la carga de archivos multimedia a través de UPnP.

-D , --disallow-deletion

Los clientes UPnP no pueden eliminar archivos multimedia en el servidor.

-g , --log-nivel= NIVEL

Lista separada por comas de pares DOMINIO : NIVEL , que permite establecer el nivel de registro individualmente para cada dominio, donde DOMINIO es "*" , " rygel " o el nombre de un complemento. Los niveles de registro permitidos son 1=crítico, 2=error, 3=advertencia, 4=mensaje/información, 5=depuración.

-u , --plugin-ruta= RUTA

Establezca la ruta de búsqueda del complemento.

-e , --ruta-motor= RUTA_MOTOR

Configure la ruta de búsqueda del motor de transcodificación y transmisión de medios.

-d , --disable-plugin= NOMBRE_DEL PLUGIN

Deshabilite el complemento PLUGIN_NAME .

-i , --title= PLUGIN_NAME : TÍTULO

Establecer el título del complemento PLUGIN_NAME en TÍTULO Puede usar esta opción más de una vez para cada complemento cuyo nombre desee establecer.

-o , --plugin-option= PLUGIN_NAME : OPCIÓN : VALOR1[,VALOR2,...]

Establecer opciones para un complemento.

-P , --disable-upnp

Deshabilite la publicidad a través de UPnP y configure rygel en modo de solo transmisión.

-c , --config= ARCHIVO_CONFIGURACIÓN

Utilice CONFIG_FILE en lugar de ${XDG_CONFIG_DIR}/rygel.conf.

-s , --apagar

Apague una instancia actualmente en ejecución de rygel

-r , --reemplazar

Reemplace la instancia actualmente en ejecución de rygel con esta instancia. Abreviatura de rygel -s && rygel

ARCHIVOS

/etc/rygel.conf

El archivo de configuración de todo el sistema para controlar el comportamiento de rygel. Ver rygel.conf(5) para más detalles.

${XDG_CONFIG_DIR}/rygel.conf

El archivo de configuración por usuario para controlar el comportamiento de rygel. Ver rygel.conf(5) para más detalles.

${XDG_CONFIG_DIR}/Rygel/*.xml

Archivos de descripción de dispositivo UPnP almacenados en caché.

AMBIENTE

RYGEL_DISABLE_UPNP

Deshabilite la publicidad a través de UPnP y configure rygel en modo de solo transmisión.

RYGEL_IFACE

Configure las interfaces de red a las que enlazar.

RYGEL_PORT

Defina el puerto de red al que se vinculará.

RYGEL_DISABLE_TRANSCODING

Disable transcoding globally.

RYGEL_LOG

Set the log level of rygel.

RYGEL_PLUGIN_PATH

Set the plugin search path of rygel.

RYGEL_ENGINE_PATH

Set the media streaming and transcoding engine search path for rygel.

RYGEL_MEDIA_ENGINE

Set a specific media streaming engine to pick up from the media engine search path.

RYGEL_DISABLE_UPLOAD

Disable media file upload via UPnP.

RYGEL_DISABLE_DELETION

Disable remote file deletion via UPnP.

RYGEL_PLUGIN_TIMEOUT

Set the time-out for finding the plugins. Useful to increase when running inside valgrind.

RYGEL_DATABASE_DEBUG

If set to 1, the database library will print the used SQL statements.

Also for every plugin you can set the following environment variables:

RYGEL_PLUGIN_NAME_TITLE

Set the title of the plugin.

RYGEL_PLUGIN_NAME_ENABLED

Enable or disable the plugin.
